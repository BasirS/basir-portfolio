## 🌐 How to View the Site

### **🚀 Live Website**
Visit the live portfolio at: **[basirs.github.io/basir-portfolio](https://basirs.github.io/basir-portfolio)**

### **💻 Local Development**

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BasirS/basir-portfolio.git
   cd basir-portfolio
   ```

2. **Open in your browser:**
   ```bash
   # Option 1: Direct file opening
   open index.html  # macOS
   start index.html # Windows
   xdg-open index.html # Linux
   
   # Option 2: Using Python simple server
   python -m http.server 8000
   # Then visit: http://localhost:8000
   
   # Option 3: Using Node.js live-server (if installed)
   npx live-server
   ```

3. **Development tips:**
   - The site is fully self-contained with no build process required
   - All styles and scripts are embedded for optimal performance
   - Responsive design works across all modern browsers
   - Uses modern CSS and JavaScript features (requires up-to-date browser)

### **📱 Mobile Responsiveness**
- Optimized for all screen sizes (320px and up)
- Touch-friendly navigation and interactions
- Compressed images and optimized loading
- Progressive enhancement for older devices
